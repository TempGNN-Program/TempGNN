This file records the execution environment used to reproduce the main
results of the TempGNN artifact. It includes system information, CPU/GPU/FPGA
device information, compiler and toolchain versions, Python package versions,
Git revision information, and default experimental settings.

The file is automatically generated by:
    information/collect_env.sh

The recorded environment corresponds to the platforms used in the paper:
- FPGA: Xilinx Alveo U280
- GPU: NVIDIA A100 (80 GB HBM2e)
- CPU: Intel Xeon Platinum 8357B

Default experiment settings:
- Precision: FP32
- Batch size: 1000
- Models: JODIE, TGAT, TGN, APAN
- Datasets: Wikipedia, MOOC, Reddit, LastFM, WikiTalk, GDELT

For full reproduction steps, please refer to the repository README and the
artifact run scripts under the corresponding experiment directories.
